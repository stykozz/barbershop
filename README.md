BarberCraft Supplies - Уеб платформа за бръснарски принадлежности
Описание
BarberCraft Supplies е модерен уеб проект, предназначен за управление на онлайн каталог с бръснарски принадлежности. Платформата включва функционалности за разглеждане на продукти, филтриране по категории, търсене, управление на количка за покупки, както и система за автентикация (вход и регистрация) с роли за потребители и администратори.

Използвани технологии
Проектът е реализиран като Single-Page Application (SPA), използвайки следните технологии:

HTML5 / CSS3: За структура и дизайн на потребителския интерфейс.

React.js (v18): Библиотека за изграждане на динамичен потребителски интерфейс чрез компоненти.

Bootstrap 5: CSS рамка за бързо и адаптивно оформление.

FontAwesome: За икони и визуален стил.

LocalStorage API: За персистентно съхранение на данните (продукти, отзиви, потребители) директно в браузъра на потребителя.

JavaScript (ES6+): За логиката на приложението и управление на състоянията (State Management).

Инструкции за стартиране
Тъй като проектът е разработен с цел максимална преносимост, той не изисква инсталация на допълнителни сървърни среди (като Node.js или PHP сървър).

Изтегляне: Уверете се, че имате файла index.html (или вашия основен файл с разширение .html) на вашето устройство.

Отваряне: Кликнете два пъти върху файла или го плъзнете в който и да е съвременен уеб браузър
